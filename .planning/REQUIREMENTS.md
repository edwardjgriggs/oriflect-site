## Out of Scope

| Feature | Reason |
|---------|--------|
| Client portal / login system | Not needed for a marketing site |
| E-commerce / payment processing | Services sold through discovery calls |
| CMS backend | Static site chosen for speed and simplicity |
| AI chatbot on site | High complexity, low v1 value |
| Heavy animations / parallax | Performance risk, doesn't match clean aesthetic |
| Real-time chat widget | Unnecessary complexity for a small consulting firm |
