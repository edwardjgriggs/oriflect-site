# Oriflect

## What This Is

A professional website for Oriflect, an AI consulting and implementation business that helps small and medium businesses (5-200 employees) adopt and integrate AI tools into their operations. The site serves as the primary conversion funnel — building trust and driving visitors to book a discovery call.

## Core Value

Visitors understand what Oriflect does, trust its expertise, and book a discovery call — the site converts interest into conversations.

## Context

Oriflect fills the gap for SMBs that know they need AI but lack in-house expertise. The business model has three layers:

1. **AI Discovery Audit** ($500–$2,000) — Entry point. Analyze client workflows, map AI opportunities, deliver prioritized recommendations with ROI projections.
2. **AI Implementation** ($2,000–$10,000) — Core revenue. Select tools, build automations, create custom prompts/templates, train staff. Common projects: chatbots, email/marketing automation, AI-powered CRM, document processing.
3. **Staff Training** — Enablement layer that ensures adoption sticks.

The audit is the wedge that leads to implementation work. The website needs to make this pipeline clear and compelling.

Brand assets (logo, colors, fonts) are ready. Content (copy, case studies) needs to be created as part of this build.

## Constraints

- **Tech stack**: Static site — fast, simple, cheap to host
- **Timeline**: ASAP — ship fast, iterate later
- **Content**: No existing copy — needs to be drafted during build
- **Booking**: Must support both contact form and Calendly embed
- **Design**: Professional and clean — corporate-leaning but not stuffy. Think McKinsey meets tech.
