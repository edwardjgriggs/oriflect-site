  <name>Task 1: Scaffold Astro project with Tailwind v4 and design system</name>
  <files>package.json, astro.config.mjs, tsconfig.json, src/styles/global.css, public/favicon.svg, src/assets/logo.svg</files>
  <action>
1. Create the Astro project using `npm create astro@latest -- --template minimal --no-install` in the current directory (not a subdirectory). If the command requires a project name, use `.` to scaffold in-place.

2. Install dependencies:
```bash
npm install @tailwindcss/vite tailwindcss
npm install @fontsource-variable/montserrat @fontsource-variable/inter @fontsource-variable/playfair-display @fontsource-variable/jetbrains-mono
```

3. Configure `astro.config.mjs`:
```javascript
import { defineConfig } from 'astro/config';
import tailwindcss from '@tailwindcss/vite';

export default defineConfig({
  site: 'https://oriflect.com',
  vite: {
    plugins: [tailwindcss()],
  },
});
```

4. Create `src/styles/global.css` with the complete design system using Tailwind v4 CSS-first config:
- `@import "tailwindcss";`
- `@custom-variant dark (&:where(.dark, .dark *));`
- `@theme` block with all brand colors: sapphire (#0A2463), gold (#D4A843), ember (#E8651A), charcoal (#1C1C1E), ivory (#F8F6F0), sapphire-tint (#E8EDF5), gold-tint (#F5EDD8)
- Dark mode colors: dark-bg (#0F172A), dark-surface (#1E293B), dark-text (#E2E8F0)
- Font families: --font-heading ("Montserrat Variable"), --font-body ("Inter Variable"), --font-accent ("Playfair Display Variable"), --font-code ("JetBrains Mono Variable")
- Do NOT create a tailwind.config.js file (Tailwind v4 uses CSS-first config exclusively)

5. Create a simple SVG placeholder for `public/favicon.svg` (a circle or square with the sapphire color).

6. Create a text-based logo placeholder at `src/assets/logo.svg` -- just the word "Oriflect" in white text, sized for the nav bar. This will be replaced when the real logo is provided.

IMPORTANT: Do NOT use `@astrojs/tailwind` integration (deprecated for v4). Do NOT create `tailwind.config.js`. Do NOT use `@tailwind` directives.
